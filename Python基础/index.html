    0x30 动态链接库

    1. dynamically linked library（被动态连接的库），一个二进制文件，不主动运行，由其他程序调用执行。
    2. windows：dynamic link libraries（DLL）动态链接库
    3. Linux：shared objects（SO）共享库
    4. ctypes提供三种方法调用动态链接库：cdll()、windll()、oledll()
