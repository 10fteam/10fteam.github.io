    MSF域渗透

    MSF几个设置参数

    1. set ExitOnSession false #接收session后保持监听，防止假死或假session
    2. set SessionCommunicationTimeout 0 #默认情况下，session在300秒内无任何活动，就会被杀死，防止此情况则设置为0
    3. set SessionExpirationTimeout 0 #默认情况下，session在一周后强制被关闭，防止则要设置为0
    4. exploit -j -z #后台持续监听
