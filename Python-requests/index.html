    Python_requests模块

    python
    创建时间:2019-11-29 19:05
    阅读:
    1. requests模块
    2. 设定各种请求参数、请求头
    3. 发送请求
    4. 请求汇总
    5. 获取响应

    requests模块

    导入requests模块

    import requests

    设定各种请求参数、请求头

    headers={‘user-agent’:’Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/78.0.3904.108 Safari/537.36’}

    cookies={‘key’:’value’}

    allow_redirects = false #请求里可带参数，阻止重定向

    timeout=5 #请求里可带参数，设置5秒超时

    s=requests.Session() #保持会话

    proxies={‘http’:’http://10.0.0.10:12345','https':'http://192.168.1.1:23456'} #代理

    proxies={‘http’:’http://user:pass@10.0.0.10:12345/'} #需要用户密码的代理

    发送请求

    各种http方法不带参数请求网页

    import requests

    r=requests.get(‘http://www.baidu.com/')

    r=requests.post(‘http://www.baidu.com/')

    r=requests.head(‘http://www.baidu.com/')

    r=requests.put(‘http://www.baidu.com/')

    r=requests.options(‘http://www.baidu.com/')

    r=requests.delete(‘http://www.baidu.com/')

    get方法带参数请求网页

    import requests
    url=’http://www.baidu.com/'

    r=requests.get(url=url,params={‘id’:’123’})

    params={‘name’:’admin’}

    r=requests.get(url=url,params=params)

    post方法带参数请求网页

    import requests

    import json

    url=’http://www.baidu.com/'

    r=requests.post(url=url,data=json.dumps({‘id’:’1’}))

    r.json()

    post上传文件

    import requests

    url=’http://www.baidu.com/'

    files={‘file’:open(‘1.txt’,’rb’)}

    r=requests.post(url,files=files)

    请求汇总

    import requests

    url=’http://www.baidu.com/'

    params={‘username’:’admin’,’password’:’admin123’}

    headers={‘user-agent’:’Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/78.0.3904.108 Safari/537.36’}

    proxies={‘http’:’http://10.0.0.10:12345','https':'http://192.168.1.1:23456'}

    cookies={‘key’:’value’}

    r=requests.get(url=url,headers=headers,params=params,cookies=cookies,proxies=proxies,allow_redirects = false,timeout=5) #allow_redirects = false 阻止重定向，timeout=5 设置5秒超时

    s=requests.Session()

    s.auth={‘username’:’admin’,’password’:’admin123’}

    s.headers={‘user-agent’:’Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/78.0.3904.108 Safari/537.36’}

    r=s.get(url=url)

    JSON请求

    获取响应

    • r.apparent_encoding #获取网页的编码格式
    • r.encoding #获取当前的编码
    • r.text #以encoding解析返回内容
    • r.content #以字节形式（二进制）返回
    • r.headers #以字典形式返回响应头
    • r.requests.headers ##返回请求头
    • r.status_code #响应状态码
    • r.raw #返回原始响应码
    • r.ok #布尔值判断成功
    • r.json() #Requests中内置的JSON解码器，以json形式返回
    • r.raise_for_status() 抛出异常
    • r.history #返回重定向信息

    获取指定响应头

    r.headers[‘Content-Type’] #获取响应内容类型

    r.headers.get(‘Content-Type’) #获取响应内容类型

    r.cookies[‘example_cookie_name’] #获取指定cookie值
